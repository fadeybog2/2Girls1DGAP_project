# 2Girls1DGAP_project
Репозиторий семестрового проекта команды 2Girls1DGAP
Это платформер в сеттинге боулинга с возможностью настоящего сражения
с кеглями (напрыгивая на них или закидывая фаерболами)
Управление:
A - движение влево
D - движение вправо
W - прыжок
alt - фаербол
